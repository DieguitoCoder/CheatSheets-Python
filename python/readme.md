# Programa de Inventario de Tienda

Este es un programa sencillo en Python. Puedes usarlo para manejar los productos de tu tienda.

## ¿Qué puedes hacer?

- Agregar un producto
- Buscar un producto
- Cambiar el precio
- Eliminar un producto
- Ver el valor total del inventario
- Mostrar todos los productos

## ¿Cómo funciona?

El programa guarda los productos en un diccionario. Cada producto tiene:
- un nombre
- un precio
- una cantidad

## ¿Cómo usarlo?

1. Ejecuta el programa con Python.
2. El menú te mostrará las opciones.
3. Escribe los datos que se te piden (nombre, precio y cantidad).
4. El programa guardará o mostrará los productos.

## Ejemplo de uso

### Entrada (lo que escribe el usuario):


--- MENÚ DE INVENTARIO ---
1. Añadir producto
2. Buscar producto
3. Actualizar precio
4. Eliminar producto
5. Calcular valor total
6. Mostrar todos los productos
0. Salir

Selecciona una opción: 1  
Nombre del producto: Manzana  
Precio del producto: 1.5  
Cantidad disponible: 10  


### Salida (lo que muestra el programa):


Producto 'Manzana' añadido.


### Otro ejemplo:
